# Theorem Zero

A GitHub Pages ebook for *Theorem Zero: A Beginning*.

The live ebook is intended to be served from `index.html` at the root of this repository.

Core editorial rule:

> Do not change the compass opening.

The project develops a layperson reconstruction of geometry from one preserved act: fixed radius, circle, sixfold closure, equilateral triangle, triangular field, native coordinates, algebra, distance classes, triangle catalogue, ratio, area, transformation, and invariance.

## GitHub Pages

In GitHub, open **Settings -> Pages**, then set:

- Source: `Deploy from a branch`
- Branch: `main`
- Folder: `/root`

The site will publish as a GitHub Pages ebook once Pages is enabled.
